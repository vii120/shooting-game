# Shooting Game! 打擊犯罪之射擊遊戲

![home](https://i.imgur.com/S27dN3Y.png)

## Demo

https://vii120.github.io/shooting-game/

## Intro

以JavaScript所製作的九宮格射擊遊戲

好人與壞人出現的位置及時間皆為隨機變數

### Rank Record

每場遊戲都會記錄完成時間，若打敗前十名，則可輸入名字並列在排名內（record.html）

此功能使用firebase來儲存資料

### Tools

HTML：使用**Pug**撰寫

CSS：使用**SASS**撰寫
